Execution

Execution Control

Activates interaction with the execution control file. When this option is activated, the execution control file (by default, named exe_cntl in the execution directory) is checked (read) every Nstep time steps. To de-active, set Nstep = 0.

Nstep - Check execution control file every Nstep (integer, default=0) time steps.

Note: The execution directory can be changed by invoking "-e exe_cntl" command line argument.

Load Balance

Hydra provides a number of parallel run-time load-balancing options.

method - Specifies the type load balancing, method (string, default=sfc_and_hg). The value must be one of the following:

  • Recursive Coordinate Bisection - Use recursive coordinate bisection for domain decomposition,
  • Recursive Inertial Bisection - Use recursive inertial bisection for domain decomposition,
  • Space-Filling Curve - Use a space-filling curve for domain decomposition,
  • Hypergraph - Use hypergraph domain decomposition, or
  • Space-Filling Curve followed by Hypergraph - Use a space-filling curve for the initial domain decomposition, followed by a hypergraph partitioning algorithm.

diagnostics - Specify the verbosity of load balance output, diagnostics (string, default=off). Valid values are the following:

  • On - Activate diagnostic output from the load-balancer, or
  • Off - Suppress diagnostic output from the load-balancer.
