Time

Set/modify the parameters associated with the time step and the associated time integration methods. For some physics, more extensive time-step control options are available in addition to or in lieu of the basic control provided here.

nstep - Maximum number of time steps to take, nstep (integer, default=10) time steps.

term - Maximum number of time steps to take, Tf (float, default=1.0).

Note: The termination time, in units consistent with the problem definition. Tf and nstep both affect the the length of simulated time in Hydra. If the number of time steps specified using nstep would yield a simulation time greater than the termination time, the number of time steps is reduced to terminate the calculation according to Tf. Thus, Tf places a ceiling on the simulation termination time regardless of how many time steps have been specified by the nstep.

deltat - The time step value, Δt (float, default=0.01).

Note: Δt may be over-ridden by physics specific constraints on the time step.

Time Integration

Define the attributes of the time-integration method used to solve the Navier-Stokes equations. The time-step control may be either fixed time-step or using a time-step based on a fixed CFL condition.

type - Specifies the time-step control method, type (string, default=fixed_cfl). Valid values are the following:

  • fixed_cfl - Sets the time-step based on a fixed maximum CFL condition using CFLmax, or
  • fixed_dt - Uses a fixed time-step based on Δtmax.

Note: The fixed CFL time-step control uses the initial velocity field and with the initial CFL number CFL0 to estimate the time-step. For all subsequent time-steps, CFLmax is used estimate the time-step. When the time-step can increase based on CFLmax, the growth is based on the time-step scale factor α. An upper bound on the time-step is set with Δtmax.

CFLinit - Specifies the initial CFL number to use at startup with the fixed CFL time-step control, CFL0 (float, default=1.0). The initial CFL number may not exceed CFL0=100.

CFLmax - Set the maximum CFL number to use with the fixed CFL time-step control, CFLmax (float, default=2.0).

dtmax - Set the maximum time step that can be used with the fixed CFL time-step control, Δtmax (float, default=1.0).

dtscale - Factor used to increase the time-step with the fixed CFL time-step control, α (float, default=1.025).

thetaA - Time-weight for the advective terms, ΘA (float, default=0.5). By default, the time-weight uses a second-order centering in time. For explicit advection, ΘA=0.0, and for an implicit treatment, ΘA=1.0.

thetaK - Time-weight for the viscous/diffusive terms, ΘK (float, default=0.5). By default, the time-weight uses a second-order centering in time. For an explicit treatment, ΘK=0.0, and for an implicit treatment, ΘK=1.0.

thetaF - Time-weight for source terms, ΘF (float, default=0.5). By default, the time-weight uses a second-order centering in time. For an explicit treatment, ΘF=0.0, and for an implicit treatment, ΘF=1.0.

trimlast - Enable/disable the use of an exact termination time, i.e., trimming the last time step to achieve an exact termination time, trimlast (string, default=off).
