Solution Method

Set/modify the parameters associated with the solution strategy and control of non-linear iterations (when appropriate).

strategy - Specifies the solution method, strategy (string, default=projection). Valid solution methods include:

  • projection - Activates the semi-implicit projection algorithm, or
  • picard - Activates the fully-implicit, projection-preconditioned, Picard algorithm.

nitmax - Defines the maximum number of non-linear iterations to be taken during each time step, Nitmax (int, default=5).

nvec - Defines the maximum number of vectors used for nonlinear Krylov acceleration (NKA) when using the picard solution method, Nvec (int, default=0).

error_norm - When using the picard solution strategy for fully-implicit time integration, this option defines the non-linear convergence norm, error_norm (string, default=composite). Valid options for the error norm include:

  • composite - Composite RMS norm based on Gresho's norm definition, or
  • max - Use the largest dependent variable maximum norm.

eps - Define the convergence criteria for non-linear solver, ε (float, default=1.0E-4).

eps_dist - Convergence criteria for normal-distance function solved during initialization, εdist (float, default=1.0E-5).

eps_p0 - Convergence criteria for initial div-free projection and initial pressure computation performed during initialization, εp0 (float, default=1.0E-5).

drop_tol - Drop tolerance used when adding vectors to the nonlinear Krylov acceleration (NKA) subspace when using the picard solution method, εdrop (float, default=1.0E-2).

timestep_control - Activate time-step control in conjunction with the non-linear solution strategy, timestep_control (string, default=Off).

  • On - Enable automatic time-step growth/reduction, or
  • Off - Disable automatic time-step reduction and use a fixed number of iterations per time-step.

convergence - Write information about the non-linear convergence history to the conv file when using the picard solution method, convergence (string, default=Off).

  • On - Enable writing the non-linear convergence history, or
  • Off - Disable writing the non-linear convergence history.

subcycle - Subcycle the pressure solves, i.e., ignore pressure up-dates if the pressure variable has already converged, subcycle (string, default=Off).

  • On - Enable pressure subcycling, or
  • Off - Disable pressure subcycling.

diagnostics - Write diagnostics information about the non-linear convergence history to the screen when using the picard solution method, diagnostics (string, default=Off).

  • On - Enable writing diagnostics during the non-linear iteration, or
  • Off - Disable writing diagnostics during the non-linear iteration.
