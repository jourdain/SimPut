Body Forces

The specification of various body forces for the momentum equations.

name - A descriptive body force name (string, default=null). name is not used in the derived control file. A unique id for each specific body force is generated underneath the cover of the web interface. While providing a descriptive name is not required, these descriptive names clearly increase usability.

type - The type of body force (string, default=Body Force). The type must be one of the following:

  • Body Force - Specifies a body force for the momentum equation,
  • Boussinesq Force - Specifies a body force using the Boussinesq approximation to represent the buoyant forces induced by temperature, and
  • Porous Drag Force - Specifies the application of a porous drag force for the momentum equations.

Note: Specifies a body force for the momentum equation. It may be time-varying, by specifying a load curve. In addition, the body force may be prescribed for a specific element set (element block) or all element sets in the mesh.

Body Force

Specifies a body force for the momentum equation. It may be time-varying, by specifying a load curve. In addition, the body force may be prescribed for a specific element set (element block) or all element sets in the mesh.

setids - One (or more) side-set(s) on which the body force will be applied, setId (integer, required).

load_curve

A load curve allows for the specification of time-dependent body forces. The load curve is defined as a list of time and factor pairs:

  • ti The time value
  • vi The load factor

name - A descriptive load curve name (string, default=null). name is not used in the derived control file. A unique id for each specific load curve is generated underneath the cover of the web interface. While providing a descriptive name is not required, these descriptive names clearly increase usability.

time and factor pairs -

t1, v1
t2, v2
￼t3, v3
...
tNpts, vNpts

amplitude - The vector values of the body force, amplitude (3 x float, default=0.0).
