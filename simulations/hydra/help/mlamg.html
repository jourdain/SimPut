ML AMG

smoother - Specifies the smoother for the ML AMG solver, smoother (string, default=ICC). Only effective if type=AMG and amgpc=ML. Values can be one of the following:

  • ICC - Incomplete Cholesky factorization with no-fill,
  • ILU - Incomplete LU factorization with no-fill,
  • SSOR - Successive over-relaxation, or
  • CHEBYCHEV - Chebychev polynomial smoother.
